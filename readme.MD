# Expression Interpreter

## Как пользоваться программой

### Интерактивный режим:
```bash
python3 main.py
> 2+3 < 4
> false
> 5 > 3+1
> true
```

### Вычисление одного выражения

```bash
python3 main.py -e "2+3 < 4"
false
```
### Чтение из файла
```bash
python3 main.py input.txt
```
### Поддерживаемые операции:
> "+" - сложение

> "-" - вычитание

> "<" - меньше

> ">" - больше

> "=" - равно

> "!=" - не равно
